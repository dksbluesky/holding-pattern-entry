⚠ Pilot Notes — things this diagram simplifies ▾
Wind correction: this diagram ignores wind. In real flying, after entering the hold, triple the inbound drift correction angle on the outbound leg to keep from drifting outside protected airspace.

HSI course pointer: the 20° tilt shortcut only works if the course pointer is set to the inbound course. If you're in the habit of setting it to outbound, this sector picture will be backwards — double check before using it.

Timing vs. DME: if timing the outbound leg, start the clock when abeam the fix (when the TO/FROM indicator flips, or wings level after the turn) — not at fix crossing. If using DME instead, the turn point is purely distance-based, not time-based.
